# gaga
gaga is just another web framework base on go language, it's just for fun, for learn.

gaga是基于go语言的web框架，实现上参考了gin框架，请不要将其用于生产上。
它只是我用来学习web框架的实现原理而造的另一个轮子。

## Feature
- 利用前缀树实现路由匹配
- 支持动态路由
- 支持路由分组
- 支持中间件
